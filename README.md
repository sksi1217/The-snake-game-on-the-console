## Snake

# ENG
To compile the game into a `.exe` file.
```bash
g++ -o out\Snake.exe main.cpp Color.cpp Game.cpp Item.cpp Setting.cpp Snake.cpp Art.cpp -I.
```

Для компиляции игры в `.exe` файл.

```bash
g++ -o out\Snake.exe main.cpp Color.cpp Game.cpp Item.cpp Setting.cpp Snake.cpp Art.cpp -I.
```
